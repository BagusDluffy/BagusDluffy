### 👨‍💻 About Me

I am a **Freshman at Game Technology** at PENS (Politeknik Elektronika Negeri Surabaya), highly passionate about creating interactive experiences. Whether it's building robust backend web applications or designing creepy, tension-building horror games, I am always eager to learn and build something new.

- 🔭 Currently working on a **2D Platformer Horror Game** in Godot and an **Internship Monitoring System** using Laravel & React Native.
- 🌱 I’m currently diving deep into **Multiplayer Networking, Monetization Systems, and Game Engines**.
- 💬 Ask me about **Game Design, Laravel, and Web Backend**.
- ⚡ Fun fact: I love clever monetization systems and analyzing how multiplayer games retain players!
